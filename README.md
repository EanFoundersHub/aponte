# David Aponte — Sitio Web Oficial

Sitio web de David Aponte, Conferencista y Coach en Finanzas Personales y Corporativas. CEO de Financial Lab.

## Cómo publicar en GitHub Pages

1. Sube todos los archivos de esta carpeta a un repositorio de GitHub
2. Ve a Settings → Pages → Branch: main → / (root) → Save
3. En 2 minutos el sitio estará en `https://tuusuario.github.io/nombre-repo`

## Estructura

```
index.html          ← Página principal
assets/
  home/             ← Retrato, íconos, servicios, testimonios
  course/           ← Fotos del curso y módulos
README.md
```
