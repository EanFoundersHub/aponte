# Subir correctamente a GitHub Pages

1. Elimina los archivos actuales del repositorio que quedaron mezclados.
2. Extrae este ZIP en tu computador.
3. En GitHub, entra a `Add file` > `Upload files`.
4. Arrastra **todos los archivos extraídos**, no el ZIP.
5. Confirma que `index.html` tenga aproximadamente 31 KB y comience con `<!doctype html>`.
6. Confirma que también estén `styles.css`, `main.js`, `david-aponte-retrato-principal.png`, `icono-empresario.png` y `curso-foto-03.jpg`.
7. Haz el commit.
8. En `Settings` > `Pages`, selecciona `Deploy from a branch`, rama `main`, carpeta `/ (root)`.
9. Abre `https://eanfoundershub.github.io/aponte/` después de que el despliegue termine.
